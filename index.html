About Beatstream

Beatstream is a music application designed for seamless music discovery and playback. You can find songs in a year wise organized manner at home page. You can also search for years, albums, songs, playlists and artists in search bar. The unique features in this music webpage is requesting songs and sleep timer. Through the given link you can request your favorite songs. There are several options to set sleep timer too. Another unique feature is the flickering color changing effects for shuffle mode and loop mode.

Navigation Links

  • Home :Displays the main collection of music organized by years.
  • Search : Switches to the search interface where you can search for movies, songs, playlists.
  • Playlist : Shows a list of playlists.
  • Request Songs : Opens a pop up with a button as "Submit your songs" to submit song requests.
  • About : Opens a modal displaying information about the Beatstream application.

Features

  • Sleep Timer Feature : Here you see list of options to select time to turn off your music.
  • Album/Playlist Cards : Clicking on these cards navigates to the detailed view of the selected album or playlist.
  • Play Album Button : Available in the album, when it clicked it starts playing from the first song of that album.
  • Song Items : Clicking on an individual song within an album or playlist list starts playing that specific song.
  • Search Input : Used to search for albums or years or artists or playlists.

Buttons Usage

  • Shuffle Button : On or Off shuffle mode for songs shuffling. When it is in on mode then you will get color changing effect.
  • Previous Button : Goes back to the previous song in the current queue or playlist.
  • Play/Pause Button : Toggles the playback state of the current song (play or pause).
  • Next Button : Goes to the next song in the current queue or playlist.
  • Repeat Button : On or Off repeat mode for repeating current song. When it is in on mode then you will get color changing effect.
  • Progress Bar Slider : Allows users to scrub through the current song's playback.

Modal Interactions

  • OK Buttons (Timer Ended, Timer Cancelled, No Song Modals) : Closes the respective modal messages.
  • Close Buttons (Modals) : Closes any open modal dialogs (e.g., About, Request Songs, Timer Options).
  • Maximize Overlay : Clicking on the compact "Now Playing" album cover or the overlay opens the maximized player view.
  • Maximized Player Back Button : Closes the maximized player view and returns to the main application interface.
